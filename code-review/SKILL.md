---
name: code-review
description: Review code changes in the current branch and provide feedback for improvement
---
1. Inspect commits for this current branch.
2. Critically analyze code changes for the following aspects,
   - Readability & Clarity
   - Maintainability
   - Testability
   - Efficiency
   - Security
   - Consistency
   - Documentation
   - Reusability
3. Provide constructive feedback on code changes, highlighting areas of improvement and suggesting specific changes to enhance the code quality.
4. Prioritize feedback based on the impact of the code changes, focusing on critical issues first.
